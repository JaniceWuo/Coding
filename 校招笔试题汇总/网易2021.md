## 1.把一个长度为m的子序列T扩充到长度为n的S序列，要求S是字典序最小。
比如输入 5 3
        2 1 5
则输出为2 1 3 4 5
### 思考
就是遍历1到n，然后如果是T中已经有的数字就跳过。然后分别比较大小，找到合适的插入位置，最后补齐到长度为n。
### code
```java
import java.util.*;

/**
 * 字典序最小的子序列
 */
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner( System.in );
        int n = sc.nextInt();
        int m = sc.nextInt();
        boolean[] vis = new boolean[n+1];
        Queue<Integer> queue = new LinkedList<>(  );
        for(int i = 0; i < m; i++){
            int num = sc.nextInt();
            vis[num] = true;
            queue.offer( num );
        }
        StringBuilder ans = new StringBuilder(  );
        for(int i = 1; i <=n; i++){
            if(vis[i]) continue;
            while(!queue.isEmpty() && queue.peek() < i){
                ans.append( queue.poll() + " ");
            }
            ans.append( i + " " );
        }
        while (!queue.isEmpty()){
            ans.append( queue.poll() + " " );
        }
        System.out.println(ans.toString().substring( 0,ans.length()-1 ));
    }
}
```

##2.把一堆物品平分给两个同学，可以舍弃掉一些物品，要求舍弃掉的价值和最小。
例如30 60 5 15 30  舍弃掉5+15，每个同学获得60。
## code
```java
import java.util.Scanner;

public class Main2 {
    static int min = Integer.MAX_VALUE;
    static int sum = 0;
    public static void main(String[] args) {
        Scanner sc = new Scanner( System.in );
        int T = sc.nextInt();
        while(T-->0){
            int n = sc.nextInt();
            int arr[] = new int[n];
            min = Integer.MAX_VALUE;
            sum = 0;
            for(int i = 0;i < n;i++){
                arr[i] = sc.nextInt();
                sum+=arr[i];
            }
            dfs( arr,0,0,0 );
            System.out.println(min);
        }

    }
    public static void dfs(int []array, int index, int ans1,int ans2){
        if(index == array.length){
            if(ans1 == ans2){
                min = Math.min(min,sum - ans1 - ans2);
            }
            return;
        }
        int val = array[index];
        dfs(array,index+1,ans1,ans2);  //该物品舍弃
        dfs( array,index+1,ans1+val,ans2 );  //该物品给同学 A
        dfs( array,index+1,ans1,ans2+val );  //给同学B
    }
}
```
